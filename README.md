Backend: Flask

Functionality:
  Upload an image → detect lines → highlight longest lines → show edges

Features:
  - Gaussian blur
  - Canny edges
  - Hough lines
  - Line filtering by angle
  - Overlay rectangle
  - JSON response for frontend

Implementation:
  - Python
  - OpenCV
  - NumPy
  - UUID
  - Flask routes

Output:
  Processed image with detected lines + edges image
